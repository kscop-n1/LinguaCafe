# LinguaCafe 0.5.10

This release removes another legacy Vuetify 2 list shim from the vocabulary filter menus.

- Replaced the `v-list-item-group` wrappers in the vocabulary filter popups with plain Vuetify 3 list items.
- Kept the existing active-state styling and filter selection logic intact for stage, book, chapter, translation, phrases, and order-by menus.

## Validation

- Verified the vocabulary page still loads in the live browser after the rewrite.
- Verified the level filter menu opens and shows the full list of options.
- Verified `/review`, `/books`, and `/user-manual` continue to load cleanly after the rebuild.
