# LinguaCafe 0.5.9

This release removes one of the remaining legacy Vuetify 2 list shims from the theme picker while keeping the user-facing behavior intact.

## Fixed

- Rewrote the theme selection dialog to use native Vuetify 3 list items instead of the legacy `v-list-item-group` wrapper.
- Kept the theme picker behavior intact for `auto`, `light`, `dark`, and `eink`, including persisted cookie/localStorage updates.
- Preserved the active theme highlight in the picker without relying on the legacy group component.

## Validation

- Verified the dialog opens in the live browser and shows the four theme choices.
- Verified selecting `eink` still updates the stored theme and reloads as `v-theme--eink`.
- Verified `/review`, `/books`, and `/user-manual` continue to load cleanly after the rewrite.
