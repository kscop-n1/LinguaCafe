# LinguaCafe 0.5.12

This release removes two more legacy Vuetify 2 tab wrappers from user-facing screens.

## Fixed

- Replaced the home calendar popup `v-tabs-items` / `v-tab-item` wrapper with native `v-window` / `v-window-item` components.
- Replaced the admin settings layout `v-tabs-items` / `v-tab-item` wrapper with native `v-window` / `v-window-item` components.
- Kept the existing tab switching and popup toggle behavior intact in both places.

## Validation

- Verified the home calendar popup still opens and switches between the info and edit panes in the live browser.
- Verified the admin settings route still renders the tabs and switches between `Dashboard`, `Users`, and the other admin sections after temporarily promoting the live test account for verification.
- Verified `/review`, `/books`, and `/user-manual` remain clean after the rebuild.
