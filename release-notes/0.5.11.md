# LinguaCafe 0.5.11

This release migrates the user settings tabbed layout from the legacy tabs-items shim to native Vuetify 3 window items.

## Fixed

- Replaced `v-tabs-items` / `v-tab-item` in the user settings layout with `v-window` / `v-window-item`.
- Kept the account and themes tabs working with the existing `v-tabs` header.
- Preserved the current tab selection behavior across the account and theme settings panels.

## Validation

- Verified the user settings page loads in the live browser after the rewrite.
- Verified switching from the Account tab to the Themes tab still works.
- Verified the themes panel contents render correctly after the native window-item migration.
